# AD Account Checker
### Functional: ###
1. view AD Entries;
2. import Excel data;
3. comparison and validation of Excel and AD data;
4. export and copy data to Excel.
   
### Extensions: ###
- [DevExpress](https://www.devexpress.com/)

### Alternatives: ###
- [AdExplorer](https://learn.microsoft.com/en-us/sysinternals/downloads/adexplorer)
